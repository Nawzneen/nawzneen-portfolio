Project Information

  • Category: Game Development
  • Software: Blender, Substance Painter, Visual Studio Code
  • Languages: HTML, CSS, JavaScript
  • Github URL: https://nawzneen.github.io/Game-Foxy/

Every object in the game is designed in blender (3D) and then were rendered 2.5D. an sprite sheet were made out of 3D animation. the background layers are downloaded freely from https://bevouliin.com/ .

  • Summary: The player is running to the right side. at the begining the player has 3 hearts. Enemy respawn every few seconsd out of right side, and the player should avoid hitting them by jumping. as the time goes by, the game speed increases.
  • Objective: Getting the highest point.
  • Rules:
    1. If the player hits the enemy, he/she loses 1 heart
    2. If the player loses all the hearts, the game is over
    3. As the time goes by the player gains points
    4. Player can jump and get the bonuses for extra points
