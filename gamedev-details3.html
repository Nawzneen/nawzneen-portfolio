A Javascript browser game. The models are all build and rendered 2.5D in blender.

  • Summary: In Easy mode, the game is over, when the time gets to 0. in the hard mode player should avoid hitting bomb, or the last place that bomb has been. also the player can not hit the empty spaces.
  • Objective: Getting the highest point.
  • Rules:
    1. hit the mole and get 1 point each time!
    2. Hard mode: if you hit an empty space, you will lose 1 point!
    3. Hard mode: if you hit the enemy, or the last place the enemy has appeared, you will lose!
